## プログラム読取

### 問2-1

次のプログラムから、ローカル変数、グローバル変数、静的なローカル変数をそれぞれ抜き出せ。

```c
#include <stdio.h>

int var;

int main(void)
{
    int count;
    static int arc;

    return 0;
}
```

- ローカル変数
  - count (int型)

- グローバル変数
  - var (int型)

- 静的なローカル変数
  - arc (int型)
